## About

This project is a basic personal website created using HTML. It serves as a digital portfolio to showcase my skills, projects, and personal information. The website is designed to be simple, responsive, and easy to navigate.

The primary goal of this project is to provide an online presence for individuals looking to share their work and connect with others. It's a great starting point for anyone new to web development or those seeking a straightforward way to create a personal website.

The website is built using standard HTML, making it lightweight and compatible with most web browsers. No JavaScript or CSS frameworks are used to keep the project simple and easy to understand.
